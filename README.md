# Информационная система розничного магазина

Десктопное приложение для автоматизации работы розничного магазина, выполненное в рамках лабораторной работы № 6.

## Технологии
- C# .NET 8
- WPF с архитектурой MVVM
- Entity Framework Core
- Microsoft SQL Server 2022 Express
- Проектирование БД в DBForge Studio for SQL Server

## Git Flow
- `main` — стабильная версия
- `develop` — ветка разработки
- `feature/*` — ветки для новых функций

## Развёртывание
Подробные инструкции по запуску и установке находятся в файле [DEPLOY.md](DEPLOY.md).
